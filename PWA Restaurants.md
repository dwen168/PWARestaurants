# PWA Restaurants

This project is a Progressive Web Application for browsing restaurants.

## Project Structure

-   `frontend/`: Contains the frontend code (HTML, CSS, JavaScript) for the PWA.
-   `backend/`: Contains the backend server code (Node.js).
-   `database/`: Contains the database file.
-   `design/`: Contains design artifacts like architecture diagrams.

## Getting Started

### Prerequisites

-   Node.js and npm

### Backend

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    node server.js
    ```

### Frontend

Open `frontend/index.html` in your web browser.
